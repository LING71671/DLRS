# 示例档案 / Examples

本目录收录的所有档案均为**虚构示例 + 占位指针**，仅用于帮助贡献者理解 DLRS 的结构与字段，不包含任何真实或敏感信息。

> English version: see [`examples/README.en.md`](README.en.md).

## 📌 概览
本目录包含 DLRS 系统中使用的示例档案模板。

所有示例都使用**虚构数据与占位内容**，不包含任何真实或敏感信息。

## 🧠 这些示例的目的
- 帮助新手理解 DLRS 档案的结构
- 展示不同的使用场景（个人、公开、受保护数据等）
- 为创建自己的档案提供起点

## 📂 你会在每个示例目录里看到什么
- `manifest.json` 文件
- 描述档案的元数据
- 结构化的样例数据

## 💡 给贡献者的小贴士
- 保持示例简单易懂
- 仅使用虚构数据
- 通过注释解释关键字段
